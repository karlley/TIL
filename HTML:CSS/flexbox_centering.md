flexbox 使用時に要素の中央配置

## ポイント

* センタリングしたい要素の親要素に`display: flex` を指定
* 水平方向の中央寄せは`justify-content: center`
* 垂直方向の中央寄せは`align-items: center`
* 他の方法よりも一番簡単でコード少なくて済む
