# i18n_generators

i18nで使用するロケールファイルの自動生成を行うgem。

[amatsuda/i18n\_generators: A pack of Rails generators gem plugin that generates Rails 3 and Rails 2 I18n locale files for almost every known locale\.](https://github.com/amatsuda/i18n_generators)

手順

[日本語対応のためのGem 'rails\-i18n'の使い方](https://zenn.dev/atsushi101011/articles/7e81a494ce69fc)

[Rails 3\.2 i18n\_generators メモ \- Qiita](https://qiita.com/gouf/items/e33bd512550aa2219da8)

プロジェクト毎にGemfileでインストールしないと動作しない

```shell
$ gem install i18n_generators
```
