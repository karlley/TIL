# command_line

コマンドラインでの操作

## カーソル移動

* `C - f/b`: 前後
* `C - a/e`: 行頭/行末
* `Esc - f/b`: 単語単位

## 削除

* `C - h`: 後方1文字削除
* `C - w`: 1単語削除

## カット/ヤンク

* `C - k`: 行末までカット
* `C - u`: 行頭までカット
* `C - y`: ヤンク

## レスポンスが無い場合

* `C - q`: 画面ロック解除
* `C - c`: コマンド強制終了

## コマンド履歴

* `C - p/n`: 1つ前のコマンド/次のコマンド
* `C - r`: 履歴検索

## ワイルドカード

* `*`: 任意の文字列
* `?`: 任意の1文字
